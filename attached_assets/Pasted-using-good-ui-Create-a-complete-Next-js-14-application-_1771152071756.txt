## PROJECT OVERVIEW
Scanvas is a website accessibility auditing tool where users enter a URL and get:
- Accessibility score (0-100)
- List of detected issues with severity
- Visual highlights of problems
- Fix suggestions
- Export reports (JSON/PDF)
- Audit history (optional, using database)

## TECH STACK REQUIRED
- Next.js 14 (App Router)
- React 18
- TypeScript (optional but preferred)
- Tailwind CSS for styling
- axe-core for accessibility testing
- JSDOM for server-side HTML parsing
- Neon Database (Vercel Postgres) for history
- Drizzle ORM for database queries
- html2canvas + jspdf for PDF export
- file-saver for downloads
- react-hot-toast for notifications

## DATABASE SCHEMA (using Drizzle ORM)

Create lib/db/schema.ts:

```typescript
import { pgTable, serial, text, timestamp, jsonb, integer } from 'drizzle-orm/pg-core';

export const audits = pgTable('audits', {
  id: serial('id').primaryKey(),
  url: text('url').notNull(),
  score: integer('score').notNull(),
  results: jsonb('results').notNull(),
  createdAt: timestamp('created_at').defaultNow().notNull(),
  userId: text('user_id') // For future auth
});

export type Audit = typeof audits.$inferSelect;
export type NewAudit = typeof audits.$inferInsert;'
